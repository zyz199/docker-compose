> 注：建议使用`Git Bash Here`执行以下命令

```shell script
# 创建文件夹
mkdir -p D:/soft/soft-dev/Docker
cd D:/soft/soft-dev/Docker

git clone https://github.com/zyz199/docker-compose.git
cd docker-compose/Windows
```
